# Markonah - Butter Cookies (Vite + React)

Project starter untuk landing page Markonah sesuai requirement.

## Setup

1. Install dependencies

```bash
npm install
```

2. Jalankan development server

```bash
npm run dev
```

3. Buka http://localhost:5173

## Catatan
- Credentials Contentful sudah ada di file `.env`.
- WhatsApp CTA menggunakan nomor dari `.env` dan pesan pra-terisi.
- Gunakan `contentful` npm package (sudah ditambahkan di `package.json`).
